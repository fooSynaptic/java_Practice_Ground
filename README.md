## 🚀 Getting Started

### Prerequisites
- Java JDK 13 or higher
- Eclipse or any Java IDE (optional)

### Running Examples

Each curriculum folder is a self-contained Eclipse project. To run:

1. **Using Eclipse:**
   - Import the project folder into Eclipse
   - Run `Main.java` as Java Application

2. **Using Command Line:**
   ```bash
   cd curriculums/<topic>/src
   javac com/itranswarp/learnjava/Main.java
   java com.itranswarp.learnjava.Main
   ```
